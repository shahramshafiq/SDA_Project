Fully Dressed Use Case - Process Natural Language Input

Use Case Name:                    Process Natural Language Input
Scope:                            INNsight Hotel Booking System
Level:                            User-Goal
Primary Actor:                    AI Receptionist System
Stakeholders and Interests:       Customer, AI Receptionist System

Main Success Scenario
Actor Action                                    | System Response
------------------------------------------------|------------------------------------------------
Customer sends a text message                   | System receives the message
                                                | System breaks the message into words/tokens
                                                | System cleans and normalizes the text
                                                | System forwards the processed input for intent detection

Extensions
1. Message is too long -> System trims or summarizes the input before processing
2. Message contains unsupported characters -> System filters them out and continues

Pre-conditions
Customer has sent a message through the chatbot interface

Post-conditions
The message has been processed and is ready for intent interpretation
