Fully Dressed Use Case - Initiate Conversation

Use Case Name:                    Initiate Conversation
Scope:                            INNsight Hotel Booking System
Level:                            User-Goal
Primary Actor:                    Customer
Stakeholders and Interests:       Customer, AI Receptionist System

Main Success Scenario
Actor Action                                    | System Response
------------------------------------------------|------------------------------------------------
Customer opens the chatbot interface            | System displays a welcome message
Customer sends a greeting message               | System responds with a greeting
Customer indicates they need assistance         | System asks how it can help

Extensions
1. System is unavailable -> Customer sees an error message
2. Customer sends an empty message -> System prompts customer to type something

Pre-conditions
The INNsight Hotel Booking System is accessible and running

Post-conditions
A conversation session has been started between the customer and the system
