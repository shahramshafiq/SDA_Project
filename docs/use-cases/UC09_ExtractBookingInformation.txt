Fully Dressed Use Case - Extract Booking Information

Use Case Name:                    Extract Booking Information
Scope:                            INNsight Hotel Booking System
Level:                            User-Goal
Primary Actor:                    AI Receptionist System
Stakeholders and Interests:       Customer, AI Receptionist System

Main Success Scenario
Actor Action                                    | System Response
------------------------------------------------|------------------------------------------------
System identifies the user's intent             | System scans the message for relevant details
                                                | System extracts room number or type
                                                | System extracts guest name and dates
                                                | System extracts booking ID if present
System compiles the extracted data              | Data is forwarded for validation

Extensions
1. Key details are missing -> System asks the customer to provide the missing information
2. Conflicting details found -> System asks the customer to clarify

Pre-conditions
User intent has been identified (booking or cancellation)

Post-conditions
Structured booking data has been extracted from the customer's message
